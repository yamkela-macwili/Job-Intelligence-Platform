# PRODUCT REQUIREMENTS DOCUMENT (PRD)

---

## 1.1 Product Overview

The **AI-Powered Career & Job Intelligence Platform** is a web-based system that analyzes a user’s CV against a target job role and provides **data-driven, actionable insights** to improve employability.

Unlike traditional job boards, this platform focuses on:

> **“Why you are not getting hired and exactly how to fix it.”**

The system leverages AI, NLP, and structured data processing to transform raw CVs into **career intelligence**.

---

## 1.2 Objectives

### Primary Objectives

* Provide **instant CV-to-job analysis**
* Identify **skill gaps and mismatches**
* Generate **personalized career improvement plans**
* Improve job application success rates

### Secondary Objectives

* Build a **scalable AI-driven backend**
* Enable **data persistence for performance optimization**
* Lay the foundation for **career analytics and insights**

---

## 1.3 Target Users

### Primary Users

* University students and graduates
* Entry-level job seekers
* Self-taught developers / career switchers

### Secondary Users (Future Scope)

* Recruiters
* Career coaches
* Training platforms

---

## 1.4 Problem Statement

Job seekers face several key challenges:

* No feedback after job applications
* Poor alignment between CVs and job descriptions
* Lack of awareness of required skills
* No structured path to becoming job-ready

---

## 1.5 Solution

The platform provides:

* CV parsing and structured extraction
* AI-powered job matching
* Skill gap analysis
* Personalized career roadmap
* CV improvement recommendations

---

## 1.6 Core Features

### 1. CV Upload & Parsing

* Accept PDF uploads
* Extract structured text using NLP tools

---

### 2. Job Role Input

* User selects or inputs a job description
* Supports predefined roles (e.g., Data Analyst)

---

### 3. AI Job Match Engine

* Semantic comparison (not keyword matching)
* Generates:

  * Match score (0–100)
  * Alignment insights

---

### 4. Skill Gap Analysis

* Identifies:

  * Missing skills
  * Weak competencies

---

### 5. Career Roadmap Generator

* Step-by-step improvement plan
* Time-based progression (e.g., 2–4 weeks per skill)

---

### 6. CV Optimization Engine

* Suggests improvements:

  * Bullet point rewriting
  * Keyword alignment
  * Impact-focused phrasing

---

### 7. Analysis Persistence (Critical Feature)

* Stores:

  * CV content
  * Analysis results
* Enables:

  * Faster responses
  * Reduced AI calls
  * Future analytics

---

## 1.7 Functional Requirements

* Upload and process CV files (PDF)
* Extract and clean text data
* Accept job description input
* Perform AI-based analysis
* Store and retrieve analysis results
* Return structured JSON responses
* Render results in dashboard UI

---

## 1.8 Non-Functional Requirements

* Response time < 3 seconds (optimized via caching)
* High reliability during demo
* Scalable architecture
* Secure file handling
* Clean, intuitive UI

---

## 1.9 Success Metrics

* Number of completed analyses
* Response latency
* Accuracy of skill extraction
* User engagement with insights
* Demo success (clarity + impact)

---

## 1.10 Future Scope

* User authentication & dashboards
* AI career coach (chat-based)
* Job recommendation engine
* Learning platform integrations
* Labor market analytics
